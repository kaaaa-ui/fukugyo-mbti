# エルメ キーワード応答設定（MBTI診断 16タイプ）

診断結果ページで「続きをLINEで見る」をタップすると、
LINEチャットが開いて「タイプコード+診断結果」（例: ESTJ診断結果）が自動入力される。
ユーザーが送信すると、以下の自動応答が返る。

## 設定方法
エルメ管理画面 → 自動応答 → キーワード応答 → 新規作成
- **キーワード**: 完全一致で設定（例: `ESTJ診断結果`）
- **応答タイプ**: テキスト
- **アクション**: タグ「MBTI診断済み」を付与
- 通常アンケートの配信条件に「MBTI診断済みタグなし」を追加
